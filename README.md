# PMS 生产管理系统 v2.0

基于 Vue3 + FastAPI 的完整生产工单流程管理系统

## 核心功能

### 1. 工单管理（核心）
- **创建工单** - 选择工单类别自动配置工序步骤
- **下发工单** - 计划员审核后将工单下发到产线
- **运行工单** - 各工序步骤登记（操作员、工时、产量、工作量比例）
- **工单流转** - 线路板→半成品→成品→检验→发货 的完整流程
- **工单进度** - 可视化展示每个步骤的状态（待开始/进行中/已完成）

### 3. 工单类别
- 线路板工单：补焊 1 → 补焊 2 → 补焊 3
- 半成品工单：补焊 → 装配 → 调试 1 → 调试 2 → 老化
- 成品工单：补焊 → 装配 → 调试 1 → 调试 2 → 老化 → 送检 → 检验 1 → 检验 2
- 检验工单：送检 → 检验 1 → 检验 2
- 发货工单：核对 → 装箱 → 物流登记
- 维修工单：维修登记 → 维修处理 → 维修完成
- 售后工单：售后登记 → 售后处理 → 售后完成

### 4. 步骤登记
- 支持多人协作登记同一工序
- 工作量比例调节（默认 1.0 均分，可调高/低）
- 自动计算工时和产量
- 步骤完成后自动流转到下一工序

## 技术栈

**前端**
- Vue 3 + Composition API
- Element Plus UI
- Vue Router（带权限守卫）
- Axios
- ECharts

**后端**
- FastAPI
- SQLAlchemy ORM
- SQLite 数据库
- Pydantic 数据验证

## 主要页面

1. **工作台** (`/dashboard`) - 数据统计、待办工单
2. **工单管理** (`/work-orders`) - 工单列表、创建、详情、下发
3. **步骤登记** (`/step-register`) - 工序步骤登记入口
4. **物料管理** (`/materials`) - 物料 CRUD
5. **发货管理** (`/delivery`) - 发货单管理

7. **统计报表** (`/reports`) - 产量、工时统计

## 工单流程示例

```
创建工单 (计划员)
    ↓
待下发
    ↓
下发工单 (计划员)
    ↓
已下发 → 步骤 1 登记 (焊接工)
    ↓
步骤 2 登记 (装配工)
    ↓
步骤 3 登记 (调试工)
    ↓
...
    ↓
全部步骤完成 → 已完成
```
