## 🎯 一键部署（只需点击一个链接）

### 点击下面的链接自动部署：

**[👉 点击这里部署到 Railway](https://railway.app/template/new?repo=https://github.com/xiabin123/pms-system)**

### 部署步骤：

1. 点击上面的链接
2. 用 GitHub 账号登录（如果还没登录）
3. 选择 `xiabin123/pms-system` 仓库
4. Railway 会自动识别并部署前后端
5. 等待 2-3 分钟，部署完成！

### 部署完成后：

Railway 会给你两个公网 URL：
- **前端地址**：类似 `https://pms-frontend-production.up.railway.app`
- **后端 API**：类似 `https://pms-backend-production.up.railway.app`

**把前端地址发给别人，他们就可以访问你的系统了！** 🎉

## 🔧 如果自动部署有问题

### 手动部署（也很简单）：

1. 访问：https://railway.app
2. 登录 GitHub 账号
3. 点击 **New Project**
4. 选择 **Deploy from GitHub repo**
5. 选择 `xiabin123/pms-system`
6. Railway 会自动识别并部署

## 🎯 部署后需要做什么？

### 1. 修改前端 API 地址

部署完成后，编辑 `frontend/src/api/request.js`：

```javascript
// 把 localhost 改为你的 Railway 后端地址
const BASE_URL = 'https://你的后端地址.up.railway.app'
```

然后提交代码，GitHub Actions 会自动重新部署。

### 2. 测试访问

- 打开前端 URL
- 使用测试账号登录（见 README.md）
- 确认功能正常

### 3. 分享给别人

把前端 URL 发给你的同事/朋友，他们就可以访问了！
